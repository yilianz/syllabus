COURSE DESCRIPTION: Formal specification syntax and semantics; structure of algorithms; list processing and string manipulation languages; statement types, control structures, and interfacing procedures.

PREREQUISITES:  CSCI A220 with a grade of C or better.

Education Resource:

COURSE OBJECTIVES: This course is an introduction course to structure and design of programming languages. It covers basic of the theory of programming languages, and trends in this field. The course will also provide tools to understand unfamiliar languages.
Students are expected to

  • Learn and understand the formal method of the description of programming language
  • Develop the skill to evaluate existing and future programming language
  • Write well-structured program with less grammar error and be a skillful debugger
  • Be able to choose programming language for a given task based on suitability
  • Prepare for future study of new programming language
