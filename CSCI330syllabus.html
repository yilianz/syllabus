COURSE DESCRIPTION:  Introduction to structure and design of programming languages. The course will discuss the organization of current programming language and the difference. The course will also provide some basic introduction on writing programs of representative.

PREREQUISITES:  Grade of C or better in ACSC210 and ACSC220 or consent of instructor.

TEXTBOOK:

  • Concepts of Programming Languages(8th Edition), by Robert W. Sebesta.

COURSE OBJECTIVES:

  • Learn and understand the formal method of the description of programming language
  • Develop the skill to evaluate existing and future programming language
  • Write well-structured program with less grammar error and be a skillful debugger
  • Be able to choose programming language for a given task based on suitability
  • Prepare for future study of new programming language

COURSE ORGANIZATION & TOPICS TO BE COVERED: This course will emphasize interactive learning.   Discussions in class  are strongly encouraged. 

Topics include:  Syntax and semantics, names, Scoping and binding, data types, expressions, control structure, subprograms,  imperative, functional, logic-based, and object-oriented programming, exception handling
