COURSE DESCRIPTION: This course focuses on the architectural design and engineering of modern web systems to develop scalable, enterprise-ready solutions. Core concepts include modern architectures: Single-Page Applications (SPA), Serverless and Function-as-a-Service (FaaS) Architectures, and RESTful Web APIs. The course covers both leading frontend and backend frameworks, and introduces principles for integrating modern applications with existing enterprise systems. Practical experiences in project deployment and security discussions on commercial cloud platforms are also included.

PREREQUISITES:  Graduate standing or permission of instructor. (Proficiency in basic HTML/CSS/Programming assumed).

TEXTS & RESOURCES:

  • Resource: MDN Web Docs

COURSE OBJECTIVES:

This is an advanced, project-based course designed for graduate students. The pace will be significantly faster than undergraduate courses, with a focus on architecture, scalability, and cloud integration. At the end of the class, students will:
  • Design and implement full-stack Single Page Applications (SPAs) using TypeScript and modern frameworks (React.js).
  • Implement scalable backend solutions using Node.js, Cloud Functions, and NoSQL databases.
  • Implement CI/CD pipelines using GitHub Actions to auto-deploy projects to Firebase/AWS.
  • Identify and mitigate common web vulnerabilities in both frontend and backend code.

COURSE ORGANIZATION and POLICY: This is a project-based course. Students will work on weekly or multi-week projects that build upon one another. For each major milestone, students will give a short presentation, receive feedback, and iterate on their product.

Students are expected to code extensively outside of class. As a graduate course, you are expected to research documentation and solve implementation challenges independently before seeking guidance.

Attendance is mandatory. Students are expected to participate in discussions and peer reviews during presentations.
