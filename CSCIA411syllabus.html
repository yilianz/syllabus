COURSE DESCRIPTION:  Basic concepts and terminology of operating systems, process implementation, synchronization, memory management, protection, resource allocation, system modeling, pragmatic aspects, and case studies.

PREREQUISITES:  Grade of C or better in CSCI A220.

TEXTBOOK:

  • Operating System Concepts(10th Edition), by Silberschatz, Galvin, and Gagne.
  • Reference book: Windows Internals 7th edition (Part 1) by Pavel Yosifovich, Alex Ionescu, Mark Russinovich and David Solomon.
  • Reference book: Troubleshooting with the Windows Sysinternals Tools by Mark Russinovich, Aaron Margosis.

COURSE OBJECTIVES:

  • Understand the fundamental concepts, structure, and mechanisms of modern operating systems.
  • Be able to write program to simulate the underlying model used by modern operating systems.
  • Identify the complexity of operating system and understand how operating system, user program and hardware interact.
  • Have basic knowledge of distributed operating system and know their differences from single user operating systems.
  • Be aware of  recent development of operating systems.
