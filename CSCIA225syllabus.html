COURSE DESCRIPTION: This course is an introduction to web development. The course will explore prevailing technologies in three main components of web application: client, server, and data. It will also provide practical experiences with large-scale interactive website development.

PREREQUISITES:  Grade of  C or better in CSCI145.

TEXTS:

  • Web Design with HTML, CSS, JavaScript and jQuery Set by Jon Duckett

COURSE OBJECTIVES:

 This is a one-semester introduction course to web development. Emphasis will be placed on core technologies such as HTML5, CSS3, JavaScript, Framework and server side application. Selected topic will be taught and students are expected to read extensively outside of the class.  At the end of the class, students will
  • have a fundamental knowledge of how fully functional websites work.
  • build  a solid foundation in web development.
  • be able to understand the challenge and limitation of web programming.

COURSE ORGANIZATION and POLICY: Course materials are organized based on weeks. New materials such as lecture notes, walk through demos, assignments will be posted at the beginning of each week.  Coding is the focus of this course and the best kind of learning is by doing. My advice to you is to dedicate at least three or four blocks of time every week to work on this course. You might start with lecture notes, and then follow the examples afterwards. Once you feel that you have a good grasp of the contents, you can work on the weekly assignment.

CSCI225 is an online course in which practically all of the content and interactions will be accessed through the Blackboard course. Online course offerings are expected to be 100% synchronous. Attendance is mandatory. And students are expected to participate in discussion, work on in-class activities, answer polls.
